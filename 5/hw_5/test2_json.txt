Задача №2: проверить JSON на Well formed:

{

	"surname": "Иванов"

	"name": "Иван"

	"patronymic": "Иванович"

	"birthdate": "01.01.1990"

	"birthplace": "Москва"

	"phone": "8 926 766 48 48"

}


Исходный вариант без запятых после элементов(ключ: значение) объекта. 
Коллекция должна быть заключена в [] - скобки, а объекты в ней в {} - скобки.
Но пока один массив в коллеккции, можно и без [].

[
	{

		"surname": "Иванов",

		"name": "Иван",

		"patronymic": "Иванович",

		"birthdate": "01.01.1990",

		"birthplace": "Москва",

		"phone": "8 926 766 48 48"

	}
]


Еще не уверенна насчёт даты и номера, судя по статье на Habr, 'Строки берем в кавычки, числа нет', 
но учитывая, что в обоих случаях присутствует сепаратор, можно предположить, что это строчный тип данных...
Можно было бы разбить на подмассивы с числовыми данными без кавычек.


[
	{

		"surname": "Иванов",

		"name": "Иван",

		"patronymic": "Иванович",

		"birthdate": {
			"day": 1,
			"mounts": 1,
			"year": 1990
			},

		"birthplace": "Москва",

		"phone": {
			"location": 8,
			"operator": 926,
			"number": 7664848
			}
	}
]
